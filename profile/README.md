# Mission Robotics

The San Francisco space where robots get built. Host of The Embodied Metal Hackathon — July 17.

## Start here

[missionrobotics.ai/hackers](https://missionrobotics.ai/hackers) — the course. One guided path through the whole loop: drive the arm, record demonstrations, upload them, fine-tune a model, then call the model you trained and watch it move. Work it top to bottom and it points you to the kit and tooling below when you need them.

## SO-101

Two repos, two jobs:

- [so100-hackathon](https://github.com/mission-robotics-ai/so100-hackathon) — the kit. One `pixi install` sets up the SO-101 driver, calibration, teleop, recording tools, and the kit's local course site.
- [newt-starter-so101](https://github.com/new-theory-research/newt-starter-so101) — the deploy client you build with once your model is live.

API keys, the newt SDK, and reference docs live at [newtheory-docs.vercel.app](https://newtheory-docs.vercel.app) — the kits link you there when you need them.
